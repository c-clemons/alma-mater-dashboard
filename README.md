# Alma Mater Financial Dashboard

A comprehensive financial planning and analysis dashboard for Alma Mater Inc.

## Features

- **Management Dashboard**: Executive summary with 2025 actuals and 2026 projections
- **Assumptions**: Configure all model parameters (pricing, COGS, CAC, etc.)
- **Team Tracker**: Manage team members with automatic payroll burden calculations
- **OpEx Tracker**: Track operating expenses by category
- **Wholesale Tracker**: Manage wholesale deals and pipeline
- **Export**: Download reports and data
